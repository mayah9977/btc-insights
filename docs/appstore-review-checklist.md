# App Store Review Checklist (Korea)

## 1. 앱 성격 명확화
- [x] 투자 자문 아님 명시
- [x] 수익 보장/예측/추천 표현 제거
- [x] 데이터 시각화·정보 제공 서비스로 정의

## 2. 결제 정책
- [x] 결제 = 디지털 서비스 이용권 명시
- [x] 투자 상품/금융 상품 아님 명시
- [x] 환불 정책 명시

## 3. VIP 기능
- [x] VIP = 기능 접근 권한
- [x] 성과·수익과 무관함 명시

## 4. 알림 / 실시간 기능
- [x] “신호”, “매수/매도” 표현 제거
- [x] “조건 감지”, “시장 참고 지표”로 대체
- [x] 알림에 면책 문구 포함

## 5. Extreme / 고급 기능
- [x] 진입 전 경고 문구
- [x] 사용자 확인 절차
- [x] 기본 비활성 상태

## 6. Web 연동
- [x] App은 안전 기능만 포함
- [x] 고급 기능은 Web 제공
- [x] App 내 외부 결제 링크 없음

## 7. 법적 고지
- [x] 이용약관 제공
- [x] 면책조항 제공
- [x] Footer / Settings 고지
