# Web / App 기능 분리 설계

## App (App Store)
- 실시간 데이터 시각화 (기본)
- 조건 감지 알림 (저강도)
- VIP1~2 제한 기능
- 안전한 UI 표현만 사용

❌ 제외
- 공격적 실시간 기능
- 실험적 Extreme 기능
- 상세 분석 리포트

## Web
- 고급 데이터 분석
- VIP3 / Extreme 기능
- 실험적 시각화
- 상세 히스토리 / 리포트

## App 내 Web 유도 문구
“일부 고급 분석 기능은 웹 버전에서 제공됩니다.”

※ App 내 외부 결제 링크 금지
※ 로그인 연동만 허용
